# VMware Web Hosting Project

## Project Overview

This project demonstrates the deployment and hosting of a static website on an Ubuntu Virtual Machine using VMware Cloud Director.

The virtual machine was provisioned inside VMware Cloud Director, configured with Ubuntu 22.04 LTS, and Apache2 was installed to host an HTML website.

---

## Objectives

* Create a virtual machine in VMware Cloud Director.
* Configure Ubuntu Linux.
* Install Apache Web Server.
* Host a static HTML website.
* Configure networking and firewall rules.
* Access the website through a public IP address.

---

## Technologies Used

* VMware Cloud Director
* Ubuntu 22.04 LTS
* Apache2 Web Server
* HTML & CSS
* Linux Commands
